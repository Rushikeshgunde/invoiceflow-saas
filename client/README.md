# InvoiceFlow SaaS

A modern GST Invoice & Billing SaaS built using:

- React.js
- Vite
- Bootstrap
- Material UI
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Razorpay

## Features

- GST Invoice
- Customer Management
- Product Management
- Reports
- Expense Tracking
- Subscription Billing
